# CodeAlpha Cyber Security Internship

This repository contains the tasks completed as part of my Cyber Security Internship at CodeAlpha.

## Tasks Completed

### Task 1: Phishing Awareness Training
- Developed a phishing awareness training presentation.
- Covered different types of phishing attacks.
- Explained phishing email red flags and fake websites.
- Discussed social engineering techniques used by attackers.
- Included real-world phishing case studies.
- Presented cybersecurity best practices and incident response strategies.

### Task 2: Secure Coding Review
- Performed a security audit of a Python Flask login API.
- Used Bandit (static analyzer) and manual code inspection to identify vulnerabilities.
- Found and exploited 6 vulnerabilities: SQL Injection, weak MD5 hashing, hardcoded secret key, no rate limiting, sensitive data exposure, and insecure debug configuration.
- Demonstrated live exploits using PowerShell HTTP requests.
- Developed a fully remediated, secure version of the application.
- Documented findings, CWE/OWASP mappings, and remediation steps in a formal audit report.
